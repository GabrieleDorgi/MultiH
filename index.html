Multimídias

É uma forma de comunicação que combina diferentes formas de conteúdo, como texto, áudio, imagens, animações ou vídeo em uma única apresentação

Tag iframe

É usada para incorporar outro documento ao documento html

Sintaxe:

<iframe src="URL" height="altura em pixels" width="largura em pixels" title="Título" ></iframe >

Tag Vídeo

Utilizada para incorporar vídeos presentes na própria máquina ao documento HTML.

Sintaxe:

<vídeo poster="imagem de capa pro vídeo.formato" controls >

<Source src="video/nome.formato" type="video/formato"

<Source src="video/nome.formato" type="video/formato"

...(quantas vezes for necessária para cobrir todos os formatos disponíveis do vídeo)

</video>

Tag de Áudio

Utilizada para incorporar áudios presentes na própria máquina ao documento HTML.

É utilizada a biblioteca de áudios sem direitos autorais da plataforma Youtube

Sintaxe:

<audio preload="metadata" controls >

<source src="audio/nome.formato" type="audio/formato"

<source src="audio/nome.formato" type="audio/formato"

...(quantas vezes for necessária para cobrir todos os formatos disponíveis do áudio)

</audio>

Tag Picture

Serve para o agrupamento de várias imagens

<picture>

<source media="(max=width:500px(tamanho maximo para o redimensionamento da tela))" srcset="img/nome.formato">

<source media="(max=width:700px(tamanho maximo para o redimensionamento da tela))" srcset="img/nome.formato">

...(quantas vezes for necessária para cobrir todos os formatos disponíveis da imagem)

<img src="img/nome.formato" alt="descrição"

</picture>
